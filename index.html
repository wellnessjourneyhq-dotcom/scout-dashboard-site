Questions Coaches Ask

Before You Grab It

Does it work with GameChanger exports?

Yes — that's exactly what it's built for. Export your season stats from GameChanger the way you normally do, paste the file into the Import tab, and the views fill in.

Is it for baseball and softball?

Both. The stat categories GameChanger exports are the same for baseball and softball, so the dashboard works identically for either.

Excel or Google Sheets?

You get both versions with one purchase. The Excel file opens directly; the Google Sheets version is a one-click copy link that puts a private copy in your Drive.

Do I need to be good with spreadsheets?

No. If you can copy and paste, you can run it. There are no formulas to write and no setup to configure — the Quick Start PDF walks you through your first import in about two minutes.

Is it a one-time purchase?

Yes. $22 once. No subscription, no renewal, no locked features. Use it this season and every season you coach.

Can I use it for multiple teams?

One file is set up for one team's season. For multiple teams, keep a separate copy per team — the Google Sheets copy link makes that take about ten seconds.

What if my export looks a little different?

The dashboard finds your stats by column name, not position, so normal variations in GameChanger exports are handled automatically. If your file does something unexpected, message me and I'll get it working with you.

Is it an app I have to install?

No. It's a file — it runs inside Excel or Google Sheets, which you already have. Nothing to install, no account to create, works on Mac, PC, and in any browser via Sheets.

Can I edit the dashboard?

Your data, settings, and overrides are fully editable — team name, minimum at-bat thresholds, lineup slots, and more. The formula engine is protected so you can't break it by accident.

What happens if I get stuck?

Message me through Etsy. I built the product and I answer support personally — usually with a specific fix, not a canned reply.

No. Scout Dashboard is an independent product built to work with GameChanger exports. It is not affiliated with, endorsed by, or connected to GameChanger. GameChanger is a trademark of its respective owner.

Travel ball? Rec league? School team?

All of the above. If your team's stats live in GameChanger, the dashboard works the same — travel, rec, Little League, middle school, high school JV. Anywhere a coach gets handed an export.
