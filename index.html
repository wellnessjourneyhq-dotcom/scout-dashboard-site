Questions Coaches Ask

Before You Grab It

Does this work with GameChanger Team Manager?

Yes. Export your stats the same way you always do, then paste the CSV into the Import tab. Same process, same file format.

Do I need any special software?

No. Works in Excel 2016 or newer, or Google Sheets (completely free). No macros, no add-ins, nothing to install beyond the file itself.

Does it work for softball?

Yes — built for both baseball and softball. All the same stat columns from GameChanger apply to both.

Does it work on a Mac?

Yes, in both Excel for Mac and Google Sheets.

What if I only have a few players or pitchers?

No problem. Empty rows stay blank, and you can set minimum PA and IP thresholds in the Setup tab so part-time players don't skew your results.

Will I accidentally break the formulas?

In Excel, formula cells are fully locked. In Google Sheets, they have edit warnings. Either way, the data you paste is yours to edit — the formulas stay protected.

Can I use it for multiple teams?

One file is set up for one team's season. For multiple teams, keep a separate copy per team — the Google Sheets copy link makes that easy.

Is this affiliated with GameChanger?

No. Scout Dashboard is an independent product built to work with GameChanger exports. GameChanger is a trademark of its respective owner.
